Privacy Policy

Effective date: June 23, 2026

Student Toolkit PH is an offline-first mobile app for students. The app provides tools for class scheduling, grade calculation, study timing, daily expense tracking, optional feedback submission, and basic app usage analytics.

Data We Store

The app stores user-entered class schedules, grades, grade settings, study timer settings, and expenses on the user's device using local storage. This data stays on the device unless the user deletes it in the app, removes the app, or clears app data.

Data We Collect

If the user chooses to submit feedback, the app sends the feedback comment to the app owner through a Google Apps Script endpoint and stores it in a Google Sheet. Optional feedback fields may include name, grade level, and school. The app does not require an account and does not upload schedules, grades, expenses, or timer activity to a server.

The app may collect basic analytics events to understand which tools are used and improve the app. Analytics events do not include subject names, teacher names, grade values, expense names, expense amounts, feedback comments, student names, school names, or other user-entered personal details.

Third-Party Services

The feedback feature uses Google Apps Script and Google Sheets to receive and store user-submitted feedback. Analytics, when configured, uses PostHog to collect privacy-friendly app usage events. The current version does not include advertising, cloud sync, or third-party advertising SDKs.

If future versions add ads, analytics, or other cloud features, this policy should be updated before release.

Children's Privacy

Student Toolkit PH is intended as a general student utility. Feedback submission is optional, and users should avoid submitting sensitive personal information.

User Control

Users can delete stored data inside the app where available, or remove all stored data by uninstalling the app or clearing app data from the device settings.

Contact

For privacy questions or support, contact hiei.one.apps@gmail.com.
